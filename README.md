## Prerequisites
- Linux or Windows
- Python 3.6.8
- Pipenv 2022.6.7

## Getting Started
### Installation and setup
- Clone this repo:
```bash
git clone https://github.com/SamuelAppleby/SWiMM_DEEPeR.git
cd SWiMM_DEEPeR
```

- Install required dependencies:
#### Linux
```bash
cd launchers
.\install_pip.bat
```
#### Windows
```bash
cd launchers
chmod +x pip_install.sh (Optional)
.\install_pip.sh
```
